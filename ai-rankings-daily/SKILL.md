# AI 榜单日报 Skill

## 功能概述

自动化采集 AI 模型榜单数据，生成分析报告并存入飞书文档。

## 数据来源

- **Artificial Analysis** - 智能指数、速度、价格、上下文窗口
- **OpenRouter** - 模型使用量/热度排行
- **LMSYS Chatbot Arena** - 人类盲评 ELO 排行

## 报告模板

```
# 🤖 AI 模型榜单日报

日期：YYYY年M月D日

## 🧠 智能指数排行
[模型排名列表]

## 📚 知识可靠性
[AA Omniscience Index 排行]

## ⚡ 输出速度排行
[Tokens/秒 排行]

## 💰 性价比排行
[每百万 tokens 价格]

## 📏 最大上下文窗口

## 📈 OpenRouter 热门

## 📡 数据来源

---

*本报告由 AI 自动生成*
```

## 分析统计功能

榜单报告需要包含以下分析：

1. **趋势分析** - 哪些模型排名上升/下降
2. **性价比推荐** - 按使用场景推荐模型
3. **亮点总结** - 本期重点变化

## 飞书文档操作

### 写入限制
- **不支持 Markdown 表格** - 表格会转为列表展示
- 建议用列表形式 + 缩进表现结构

### 创建文档
```json
{ "action": "create", "title": "AI 模型榜单日报 YYYY-MM-DD" }
```

### 写入内容
```json
{ "action": "write", "doc_token": "xxx", "content": "报告内容..." }
```

## 使用流程

1. 使用 `web_fetch` 抓取榜单页面
2. 解析数据，生成报告内容
3. 调用 `feishu_doc` 创建/写入文档

## 配置

可配置项：
- 文档标题格式
- 报告详细程度
- 定时任务 cron 表达式

## 示例命令

生成今日报告并保存到飞书：
1. 抓取数据 → 2. 生成报告 → 3. 写入飞书文档
